Для начала работы необходимо в целевом Битрикс24 создать локальное приложение.
Процедура:
    Авторизоваться в Битрикс24->
    Маркет->
    Разработчикам->
    Другое->
    Локальное приложение
    Параметры локального приложения:
        Использует только API, Указать путь до обработчика авторизации (домен/core/auth.php)/
        В правах указать доступ к модулям(CRM, Пользователи, Пользователи базовый)
    Забрать полученный данные(client_id, secret_id, домен Битрикс24)
    Указать эти параметры в константах в файле /config/config.php
    Перети в браузере по домену приложения->нажать кнопку авторизация
    Примечание: Авторизация держится час
